# RESUMO DO ASSUNTO
  A interação Humano Computador começa no conhecimento do cliente, que será o usuário do sistema, pois assim é possível arquitetar uma interface em que desde as cores até linguagem permitam que a pessoa converse neuralmente. Nesse sentido, há uma série de princípios fundamentais que compõem a criação de um bom sistema sendo eles:
 
Interface: por onde o usuário irá se comunicar com o sistema de forma física ou conceitual;
 
Interação: como se dá a comunicação com o sistema;

Affordance: indicação instintiva de como algo deve ser usado
     
  Esses princípios foram baseados em uma evolução histórica que envolve uma aliança entre engenharia de software e disciplinas como psicologia e sociologia envolvento divrsos pesquisadores, entre eles Jakob Nielsen que formulou um conjunto de diretrizes para verificação da usabilidade de interfaces, uma ponte com a área psicológica. Em resumo elas se conectam a filosofia de Steve Krug no livro "Não Me Faça Pensar" de que o design deve ser autoexplicativo, uma vez que se um sistema segue as heurísticas, o usuário não precisa "parar para pensar" sobre como realizar uma tarefa; a ação torna-se intuitiva. Entre os critérios análisados estão:

Usabilidade: O foco central das heurísticas de Nielsen

Experiência do Usuário (UX): A sensação geral do usuário ao interagir

Acessibilidade: A garantia de que pessoas com diferentes habilidades possam usar o sistema

Comunicabilidade: A capacidade da interface de transmitir a lógica do design ao usuário
